Past Research Projects

  • Evaluation of Pattern Recognition Ability Based on Braille (2025): Developed a wearable ring-shaped haptic interface to transmit information via Braille-like patterns, achieving a 93.3% recognition accuracy through spatio-temporal modulation.
  • VR-based Exercise & Layup Training System (2024): Created an immersive VR environment using Azure Kinect and Unity that tracks user motion in real-time to provide feedback on basketball layup techniques, effectively improving shooting success rates.
  • Refrigerator Food Expiration Management System (2022): Implemented an automated inventory system using OpenCV-based digit recognition to identify expiration dates and a web-based platform for mobile food management.
  • Makeup Guide Smart Mirror (2021): Developed an interactive smart mirror using Raspberry Pi and MediaPipe to provide real-time facial guidelines, assisting users in following complex makeup tutorials.
  • Arduino-based Fireplace Humidifier (2021): Designed a functional humidifier using Arduino that combines water mist and LED lighting to simulate a realistic fireplace effect, merging aesthetic design with environmental sensing.
